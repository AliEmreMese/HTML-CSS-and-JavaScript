# DSA Learning Platform

This project is a web-based platform designed to help users learn and visualize various Data Structures and Algorithms (DSA). It provides interactive modules for understanding how different data structures and algorithms work.

## Features

### Data Structures
- **AVL Tree**: Visualization of AVL Tree operations.
- **Binary Search Tree (BST)**: Interactive BST operations.
- **Linked Lists**: Implementation and visualization of Linked Lists.
- **Dynamic & Static Lists**: Understanding list implementations.
- **Stacks & Queues**: Visualizing Stack and Queue operations.

### Algorithms
- **Sorting Algorithms**:
  - Bubble Sort
  - Quick Sort
  - Merge Sort
  - Selection Sort
  - Insertion Sort
- **Searching Algorithms**:
  - Linear Search
  - Binary Search
